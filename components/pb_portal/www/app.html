Network & printer

Wi-Fi network and Moonraker host/port are configured together on the setup page — changing Wi-Fi re-enters the password, so they are kept on one screen to avoid dropping the device off the network.

Wi-Fi & printer setup
Control token

A control token locks every command to clients that send it. If you set one, this browser remembers it — but Klipper/Moonraker and any other device must be given the same token or they can no longer control the heater. Clear it to return to open LAN control.
